14. 常用缩写
发布于：2019-04-11 01:46:26 标签：/ Java学习 / 访问：
  1. 版本：
    • GA版本 = General Availability = 正式发布的版本 = Release
    • RC版本 = Release Candidate = 候选版本
    • LTS = Long Term Support = 长期演进版
  2. Java基础
    • OOM = Out Of Memory
    • NPE = NullPointer Exception
    • OSGi = Open Service Gateway Initiative，开放服务网关，是Java动态化模块化系统的一系列规范
    • JUC = java.url.concurrent包
    • AQS = AbstractQueuedSynchronizer，抽象队列同步器，Java提供的底层同步工具类
    • SPI = Service Provider Interface，JDK内置的一种服务提供发现机制
    • JNDI = Java Naming & Directory Interface，Java命名目录服务
    • JMS = Java Message Service，Java消息服务
    • JTA = Java Transaction API，Java事务服务
    • JNI = Java Native Interface，Java本地接口
    • CGLIB = Code Generation Library
    • SOA = Service Oriented Architecture，面向服务架构
    • JIT = Just In Time，即时编译技术
  3. 代码分层
    • DO = Data Object，与数据库表结构一一对应，通过DAO层向上传输数据源对象
    • DTO = Data Transfer Object，数据传输对象，Service和Manager向外传输的对象
    • BO = Business Object，业务对象，可以由Service层输出的封装业务逻辑的对象
    • Query，数据查询对象，各层接收上层的查询请求，注：超过2个参数的查询封装，禁止使用Map
    • VO = View Object，显示层对象，通常是Web向模板渲染引擎层传输的对象
  4. 算法
    • DP = Dynamic Planning
